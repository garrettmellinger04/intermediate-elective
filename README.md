# intermediate-elective
intermediate elective 2 -- vernal pool vegetation and aquatic invertebrates

## General information

This repository contains data and code to explore patterns in vernal pool vegetation at North Campus Open Space.

To work with the code in this repository, you will need the following packages:

```
library(tidyverse)
library(here)
library(janitor)
```
